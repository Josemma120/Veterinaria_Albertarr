ALBERTAR es un sistema integral de gestión para clínicas veterinarias que permite administrar de manera eficiente todos los aspectos operacionales del negocio.

¿Qué hace?
El sistema proporciona un panel de control centralizado donde los diferentes usuarios (administradores, secretarios y veterinarios) pueden:

Gestionar Clientes: Registrar y mantener información de los dueños de mascotas, incluyendo datos de contacto y domicilio
Administrar Mascotas: Crear perfiles detallados de las mascotas con foto, especie, raza, sexo y peso
Organizar Servicios: Definir servicios veterinarios con duración y costo, asignando veterinarios responsables
Controlar Citas: Programar, confirmar y seguimiento de citas veterinarias con cálculo automático de horarios
Gestionar Personal: Mantener registro de empleados con información laboral, horarios y salarios
Características principales
✅ Control de acceso por roles: Diferentes permisos según el tipo de usuario
✅ Búsqueda inteligente: Filtrado rápido de clientes por teléfono
✅ Interfaz intuitiva: Diseño limpio y fácil de usar
✅ Cálculo automático: Horarios de fin de cita basados en duración del servicio
✅ Gestión de imágenes: Carga de fotos para mascotas
✅ Base de datos persistente: Información segura y sincronizada
